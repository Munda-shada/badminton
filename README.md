# Smash Club Next.js

Responsive Next.js build for a private badminton group manager.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Demo accounts

- Admin: `admin@smashclub.app` / `admin123`
- Player: `mira@smashclub.app` / `player123`
- New signup invite code: `COURT-LITE-77`

## Included flows

- Invite-only sign up and login
- Admin session creation with date, time, courts, player cap, cost, and Google Maps link
- Responsive desktop and mobile player feed
- Player RSVP buttons: In, Tentative, +1, Out
- Automatic waitlist when confirmed capacity is full
- Visible roster buckets for confirmed, waitlist, tentative, and out
- RSVP freeze exactly 1 hour before start time with a live countdown
